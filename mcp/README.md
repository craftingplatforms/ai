# MCP Servers

Model Context Protocol servers that extend AI with domain-specific tools.

Examples: query Terraform state to answer "what is deployed where?", read from a service catalog, fetch metrics from observability platforms, explore cloud costs.

MCP is vendor-neutral — the same server works with Claude Code, Cursor, GitHub Copilot, and other MCP-compatible clients.
